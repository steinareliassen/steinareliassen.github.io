Setting up Trunkfriends on OSX.

Due to Apple being strict about developers registering and signing their software "properly", install files for OSX is not provided at the moment. The aim is to have e a package that contains the Java runtime and Trunkfriends, so you only need to download one file and unpack it.

For now, you first need to install Java, then download Trunkfriends as a ZIP file, and unpack it. When Java is installed, all you need to do is to doubleclick on thie unpacked file, and Trunkfriends will start.

Go to the Java download page from Oracle, and download the installer. If you use brew, you can use it to install OpenJDK@17.
