Setting up Trunkfriends on OSX.

The installation for OSX has been tested on Intel and M2 chips, and should be architecture agnostic.

For OSX, a packaged JAR file that can be executed directly as long as you have JAVA installed, is so far what is provided. The reason for this is the difficulties of building an install binary without registering as a developer with Apple. This is not free. It may be something that will be done eventually, but now, the JAR file is the way.

When Java is installed, all you need to do is to doubleclick on thie unpacked file, and Trunkfriends will start.

Go to the Java download page from Oracle, and download the installer. If you use brew, you can use it to install OpenJDK@17.

Download Trunkfriends-macos-0.4.0.zip, and extract it. To start, simply double-click on the Trunkfriends.jar file, or, in a terminal, write
java -jar Trunkfriends-macos-0.4.0.jar
